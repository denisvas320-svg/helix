2. Требования перед настройкой

Перед подключением необходимо выполнить все проверки ниже.

2.1 Проверка версии платформы 1С

Что проверить
Используется 32-битная версия платформы 1С.
Как проверить
Открыть Диспетчер задач
Найти процесс 1С
Убедиться, что используется x32 версия
Ожидаемый результат
Запущена 32-битная платформа

2.2 Проверка .NET Framework 3.5

Как проверить
Открыть:
Панель управления → Компоненты Windows
Найти:
Microsoft .NET Framework 3.5
Корректное состояние
Установлен флажок
или отображается заполненный квадрат
Если компонент отсутствует
Активировать компонент
Подтвердить загрузку и установку
Ожидаемый результат
.NET Framework 3.5 установлен

2.3 Проверка работоспособности камеры

Действия
Выполнить онлайн-тест веб-камеры
Проверить:
Камера определяется системой
Изображение отображается
Ожидаемый результат
Камера работает вне 1С

3. Пошаговое подключение веб-камеры

Шаг 1. Вход в систему

Действия
Войти в 1С под пользователем с полными правами.
Ожидаемый результат
Доступны настройки подключаемого оборудования

Шаг 2. Создание оборудования

Путь
Настройки → Подключаемое оборудование → Веб-камеры
Действия
Открыть раздел «Веб-камеры»
Нажать Создать
Ожидаемый результат
Создана новая запись оборудования

Шаг 3. Выбор драйвера

Действия
В поле Драйвер оборудования выбрать:
Хеликс: Веб-камера
Ожидаемый результат
Выбран корректный драйвер

Шаг 4. Выбор рабочего места

Действия
Нажать Показать все
Выбрать текущее рабочее место
Нажать Сохранить
Текущее рабочее место должно быть отмечено галочкой.
Ожидаемый результат
Рабочее место привязано

Шаг 5. Открытие настроек устройства

Если окно не открылось автоматически
Выбрать устройство в списке
Нажать Настройки
Ожидаемый результат
Открыто окно параметров устройства

Шаг 6. Установка драйвера

Если окно настроек пустое
Перейти во вкладку Функции
Нажать Установить драйвер
Подтвердить установку
Ожидаемый результат
Драйвер установлен

Шаг 7. Выбор модели камеры

Действия
Выбрать необходимую веб-камеру из списка устройств.
Ожидаемый результат
Камера определилась в системе

Шаг 8. Тестирование устройства

Действия
Открыть вкладку Функции
Нажать Тест устройства
Проверить отображение изображения
Сохранить настройки
Проверка
Изображение отображается
Ошибки отсутствуют
Ожидаемый результат
Камера успешно проходит тест

4. Проверка после настройки

Рекомендуется проверить работу камеры:
На рабочем месте
На рецепции
В реальном бизнес-процессе
Итог проверки

6. Ограничения

На одном рабочем месте может работать только одна веб-камера
Одну и ту же камеру можно настроить только на одно рабочее место
